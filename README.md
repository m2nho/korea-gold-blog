# 🚀 네이버 블로그 자동화

구글 시트에 정리된 데이터를 바탕으로 네이버 블로그 글을 자동으로 작성해주는 프로그램입니다.

## 주요 기능

- **자동 로그인** — 네이버 아이디와 비밀번호를 입력하면 자동으로 로그인합니다. 캡차가 뜨면 직접 해결할 수 있도록 최대 2분간 대기합니다.
- **구글 시트 연동** — 구글 시트 URL을 입력하면 블로그에 쓸 데이터를 자동으로 불러옵니다.
- **템플릿 자동 적용** — 내 템플릿 목록에서 첫 번째 템플릿을 자동으로 적용합니다.
- **데이터 자동 입력** — 시트 데이터가 템플릿의 빈칸에 자동으로 채워집니다.
- **시세 표 자동 입력** — 시트에 정리된 시세 데이터가 표에 자동으로 입력됩니다.
- **썸네일 자동 생성** — 배경 이미지 위에 강조 문구를 넣은 썸네일을 만들 수 있습니다. 강조색과 배경 이미지를 직접 선택할 수 있습니다.
- **썸네일 자동 삽입** — 생성된 썸네일이 글 맨 위에 자동으로 들어가고, 대표 사진 등록까지 처리됩니다.
- **자동 발행** — 토글을 켜면 글 작성이 끝난 후 자동으로 발행됩니다.
- **계정 정보 저장** — 아이디와 비밀번호를 암호화하여 저장해두면 매번 입력하지 않아도 됩니다.
- **미리보기** — 글을 작성하기 전에 요약, 시세표, 뉴스, 썸네일을 탭으로 미리 확인할 수 있습니다.
- **진행 상황 표시** — 현재 어떤 단계를 진행 중인지 실시간으로 확인할 수 있습니다.
- **강제 중지** — 작업 도중 중지 버튼을 누르면 즉시 멈출 수 있습니다.
- **글자 크기 조절** — Ctrl+마우스 휠 또는 하단 ±버튼으로 화면 글자 크기를 조절할 수 있습니다.

## 사용 방법

1. 프로그램을 실행합니다
2. 네이버 아이디, 비밀번호, 블로그 ID를 입력합니다
3. 구글 시트 URL을 입력하고 **데이터 불러오기**를 클릭합니다
4. 목록에서 작성할 항목을 선택합니다
5. 필요하면 썸네일 탭에서 썸네일을 미리 생성합니다
6. **자동화 시작**을 클릭합니다
7. 작업이 완료될 때까지 브라우저를 조작하지 않고 기다립니다
